# Markdown 网站 PRD

## 1. 产品概述
这是一个面向 Markdown 学习和使用场景的网站。
它既提供 Markdown 的介绍和语法教学，也支持用户读取、编辑、预览和导出 Markdown 文件。

## 2. 产品目标
让用户在一个网站里完成：
1. 学会 Markdown 基础用法
2. 打开和阅读 Markdown 文件
3. 在线编辑 Markdown
4. 实时预览渲染效果
5. 导出为常见格式

## 3. 目标用户
- Markdown 新手
- 学生
- 程序员
- 内容创作者
- 技术文档编写者

## 4. MVP 核心功能

### 4.1 Markdown 介绍页
提供 Markdown 的基础介绍，包括：
- Markdown 是什么
- Markdown 的应用场景
- 为什么 Markdown 很适合写文档和笔记

### 4.2 Markdown 使用方法页
提供常用语法说明和示例，包括：
- 标题
- 段落
- 粗体 / 斜体
- 列表
- 引用
- 代码块
- 链接
- 图片
- 表格
- 分割线
- 任务列表

要求：
- 每种语法都有示例
- 最好同时显示“源码”和“渲染效果”
- 页面结构清晰，方便查阅

### 4.3 Markdown 在线编辑器
提供一个在线编辑区域，支持：
- 输入 Markdown 文本
- 粘贴内容
- 清空内容
- 恢复默认示例内容

### 4.4 实时预览
编辑器输入后，右侧或下方实时展示渲染结果。

要求：
- 渲染结果更新及时
- 基础 Markdown 语法显示正常
- 页面排版清晰
- 代码块和表格显示正常

### 4.5 读取 Markdown 文件
用户可以通过以下方式导入 `.md` 文件：
- 文件选择
- 拖拽上传

要求：
- 读取后自动填入编辑器
- 右侧自动更新预览
- 文件格式异常时给出提示

### 4.6 导出 Markdown 文件
支持将当前编辑内容导出为 `.md` 文件。

### 4.7 导出其他格式
第一版建议支持：
- 导出 HTML
- 浏览器打印 / 导出 PDF

### 4.8 自动保存
在本地自动保存当前编辑内容，避免刷新后丢失。

建议：
- 使用 LocalStorage
- 页面重新打开时恢复上次内容

## 5. 推荐新增功能

### 5.1 语法快捷工具栏
提供常见插入操作：
- 插入标题
- 插入粗体
- 插入代码块
- 插入链接
- 插入表格
- 插入任务列表

### 5.2 目录自动生成
根据标题自动生成 TOC 目录，方便长文导航。

### 5.3 字数统计
显示：
- 总字数
- 字符数
- 行数
- 预计阅读时间

### 5.4 深色模式
支持浅色 / 深色主题切换。

### 5.5 模板中心
提供默认模板，例如：
- README 模板
- 会议纪要模板
- 学习笔记模板
- 博客文章模板

### 5.6 一键复制 HTML
支持将渲染后的 HTML 一键复制。

### 5.7 Markdown 语法校验
对不规范语法做简单提示，例如：
- 未闭合代码块
- 链接格式异常
- 表格格式不完整

## 7. 页面结构建议
### 首页
- 产品介绍
- 功能亮点
- 快速入口

### Markdown 介绍页
- Markdown 简介
- 使用场景
- 优势说明

### Markdown 教学页
- 常见语法说明
- 示例和效果展示

### 编辑器页
- 编辑区
- 实时预览区
- 工具栏
- 导入导出按钮
- 字数统计
- 目录区域（可选）

## 8. 设计要求
- 风格简洁现代
- 学习内容和工具内容区分清晰
- 编辑器页面突出可用性
- 预览区域排版舒服
- 支持桌面端和移动端
- 新手也能快速看懂

## 9. 非功能性需求
- 首屏加载快
- 编辑输入流畅
- 导入和导出稳定
- 本地存储可靠
- 基础无障碍可用
- 页面适配移动端

## 10. 限制条件
- 第一版优先前端实现
- 不依赖复杂后端
- 文件处理尽量在浏览器本地完成
- 导出 PDF 优先使用浏览器打印方案
